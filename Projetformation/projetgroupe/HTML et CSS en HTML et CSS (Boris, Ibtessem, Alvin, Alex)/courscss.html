Cascade Style Sheet

Présentation générale

Le CSS est une page de code qui s'associe à une page HTML. Celui-ci permet de mettre en valeur la forme d'une page internet. Comme le HTML, le CSS est un langage qui nous permet de donner des indications d’affichage à un navigateur. Le CSS ne s’utilise pas tout seul, indépendamment, il ne donne aucun résultat. C’est couplé au HTML qu’il prend tout son sens. Comme montré précédemment, le HTML dis au navigateur d’afficher tel et tel élément (paragraphe, titre, liste…) et le CSS dis au navigateur d’afficher ces éléments HTML d’une certaine manière (couleur de fond, image de fond, famille de police, taille de police…)

Propriété de mise en page

background : Arrière plan

background-color : Couleur d'arrière plan

background-image : Image d'arrière plan


border : Permet de créer la catéristique d'un cadre
Exemple : border: 1px solid black , le premier indique l'épaisseur de la bordure, le deuxième son style et le dernier ça couleur.
Si on veut modifier qu'un seul côté du cadre on utilisera bottom pour le bas, left pour la gauche, right pour la droite et top pour le haut.

height : Hauteur d'un élément

width : Largeur d'un élément


color : Pour la couleur du texte par exemple

display : Permet de faire disparaître un élement comme les puces d'une liste avec la valeur none

float : Permet de retirer un élément du flux normal

font : proprété concercant le caractère de texte
Exemple : font-color pour la couleur du texte
font-size pour la taille du texte
font-family pour le style d'écriture

left : Position gauche d'un élément

right : Position gauche d'un élément

list-style-image : Permet de remplacer les puces d'une liste par des images

margin : Permet de changer les marges exterieur
Exemple : margin: 1px 3px 4px 1px , la première cote est la marge haute et les autres se lisent dans le sens des aiguilles d'une montre.
Si nous voulons changer qu'une seule marge nous utiliserons margin-top, bottom, left, right

opacity : Cela créé une transparance du fond de notre cadre ça valeur est comprise entre 0 et 1

padding : Permet de changer les marges exterieur
Exemple : padding : 1px 2px 3px 1px , la première cote est la marge du haut et les autres se lisent dans le sens des aiguilles d'une montre
Si nous voulons changer qu'une seule marge nous utiliserons padding-top, bottom, left ou right

text-align : Défini l'emlacement du texte

text-decoration : Permet de souligner du texte, rayer

text-shadow : Permet de définir un effet d'ombre au texte
